# TestPublicEvents

**Структура проекта**
```sh
└── src/
    ├── components/   # Переиспользуемые компоненты
    ├── hooks/        # Функциональные хуки
    ├── navigation/   # Конфигурация экранов приложения
    ├── screens/      # Экраны
    ├── store/        # Общий стор
    └── utils/        # Вспомогательные функции
 ```

**Установка**

`yarn && cd ./ios && pod install`

**Запуск**

`yarn start`

`yarn ios`
